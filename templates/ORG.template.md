# Organization Context (optional)

Optional shared context for teams deploying the Slipstream across an
organization. If this file is present in the working folder, Slipstream reads it
during onboarding and biases recommendations accordingly. If it is absent, the
Slipstream ignores it entirely — it is never required.

Keep this file free of anything sensitive or confidential. It is context for
recommendations, not a place for secrets. If the skill is published publicly, this
file should never be included with real organizational data.

<Tools, platforms, and frameworks that are the org standard, so the plan can bias
toward learning what people actually use here rather than generic alternatives.>

## Approved / preferred resources

<Learning platforms, subscriptions, or vendors the org already has relationships
with or a license for, so recommendations lean on what is already available.>

## Sponsorship

budget-posture question can point people at the real channel.>
